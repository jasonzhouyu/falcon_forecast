# 猛禽迁徙预测系统 (RaptorCast)

## 项目简介

猛禽迁徙预测系统是一个基于气象数据和物候信息的猛禽迁徙预测工具，旨在帮助观鸟爱好者和研究人员预测猛禽迁徙的最佳时机。该系统通过分析气象条件、地形特征和物候规律，为用户提供专业的猛禽迁徙预测报告。

## 功能特点

- **直观的 GUI 界面**：简洁美观的用户界面，易于操作
- **多站点支持**：支持多个猛禽监测站点
- **7天预测**：提供未来7天的迁徙预测
- **专业报告**：生成详细的迁徙适宜度分析报告
- **实时气象数据**：获取最新的气象数据用于分析
- **eBird API 集成**：可选集成 eBird 数据以增强预测准确性

## 系统要求

- **操作系统**：Windows 10/11
- **Python 版本**：3.12+（仅开发环境需要）
- **内存**：至少 4GB
- **网络连接**：需要互联网连接以获取气象数据

## 安装方法

### 方法一：直接运行可执行文件（推荐）

1. 从 `dist` 目录中找到 `RaptorCast_Final.exe` 文件
2. 双击运行该文件即可启动系统
3. 首次运行时会自动创建配置文件

### 方法二：从源代码运行（开发环境）

1. 确保安装了 Python 3.12 或更高版本
2. 安装依赖项：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行主程序：
   ```bash
   python raptorcast_gui.py
   ```

## 使用说明

1. **启动系统**：运行 `RaptorCast_Final.exe`
2. **选择监测站点**：从左侧下拉菜单选择您感兴趣的监测站点
3. **选择预测日期**：从下拉菜单选择要预测的日期（今天、明天、后天或未来其他日期）
4. **开始预测**：点击「🚀 开始预测」按钮
5. **查看结果**：右侧面板会显示详细的预测报告，包括迁徙适宜度评分、黄金观测窗口等信息

## 配置选项

系统首次运行时会在 `dist/config/settings.json` 文件中创建默认配置。您可以通过点击「⚙️ API 配置」按钮来修改以下配置：

- **eBird API Key**：可选配置，用于获取实时鸟类观测数据
- **搜索半径**：eBird 数据的搜索半径（默认 50km）
- **回溯天数**：eBird 数据的回溯天数（默认 5 天）

## 技术架构

### 核心模块

- **raptorcast_gui.py**：GUI 界面实现，负责用户交互和结果展示
- **algorithm.py**：核心算法模块，包含迁徙预测逻辑

### 数据流程

1. 用户选择监测站点和预测日期
2. 系统获取该站点的气象数据
3. 算法分析气象条件和物候信息
4. 生成迁徙适宜度评分和预测报告
5. 在 GUI 界面展示结果

## 依赖项

- **tkinter**：GUI 界面库
- **requests**：网络请求库
- **numpy**：数值计算库
- **openmeteo_requests**：气象数据获取库
- **requests_cache**：HTTP 请求缓存
- **retry_requests**：请求重试机制

## 常见问题

### Q: 预测结果的准确性如何？
A: 预测结果基于气象数据和物候规律，仅供参考。实际迁徙情况可能会受到多种因素影响。

### Q: 为什么需要 eBird API Key？
A: eBird API Key 用于获取实时的鸟类观测数据，可以增强预测的准确性，但不是必须的。

### Q: 系统运行缓慢怎么办？
A: 系统需要获取气象数据，首次运行或网络状况不佳时可能会较慢，请耐心等待。

### Q: 如何更新系统？
A: 当有新版本发布时，只需替换 `RaptorCast_Final.exe` 文件即可。
