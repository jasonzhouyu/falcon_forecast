## 🔍 系统概述

本系统通过耦合多源气象数据与生物阈值，预测猛禽迁徙行为。核心算法突破传统线性气象加权模型，引入：

- 大气层结稳定性分析
- 侧风偏航向量计算
- 物种特异性气象响应函数
- eBird 实证数据融合

**核心创新：**
- 🏔️ 地形动力升力场建模
- 🌡️ 热力-湍流交互算法
- 🌀 逆温层顶盖效应检测
- 🧭 侧风偏航补偿逻辑
- 📡 eBird 空间异质性修正

## 🚀 快速开始

### 环境配置

1. **克隆仓库**
   ```bash
   git clone https://github.com/jasonzhouyu/falcon_forecast.git
   cd falcon_forecast
   ```

2. **创建虚拟环境**
   ```bash
   # Windows
   python -m venv .venv
   .venv\Scripts\activate
   
   # macOS/Linux
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. **配置环境变量**
   - 在项目根目录创建 `.env` 文件
   - 添加以下内容（替换为您的实际 API 密钥）：
   
   ```env
   # eBird API 配置
   EBIRD_API_KEY=your_ebird_api_key_here
   EBIRD_SEARCH_RADIUS=50
   EBIRD_BACKLOOK_DAYS=5
   ```
   
   **获取 eBird API 密钥：**
   - 访问 [eBird API 页面](https://ebird.org/api/keygen)
   - 登录您的 eBird 账号
   - 生成 API 密钥

### 依赖安装

```bash
# 安装核心依赖
pip install flask requests openmeteo-requests requests-cache retry-requests python-dotenv numpy

# 验证安装
pip list
```

### 启动应用

```bash
# 启动 Web 应用
python app/app.py

# 应用将运行在 http://127.0.0.1:5001
```

## 使用说明

1. **访问应用**
   - 打开浏览器，访问 http://127.0.0.1:5001

2. **选择监测站点**
   - 从下拉菜单中选择一个监测站点（共 8 个站点）
   - 站点包括：都统岩、龙泉山、尧山电视台、冠头岭、九龙山、渔洋山、南汇东滩、崇明东滩

3. **选择日期**
   - 从下拉菜单中选择一个日期（未来 7 天）

4. **获取预测**
   - 点击"获取预测"按钮
   - 系统将生成详细的预测报告

5. **查看结果**
   - 查看预测报告，包括：
     - 迁徙适宜度总分
     - 黄金观测窗口
     - 小时预测数据
     - 天气专项分析
     - 专业观测策略

## 技术架构

### 核心模块
- **数据获取**：Open-Meteo API 气象数据、eBird API 观测数据
- **预测模型**：地形动力学模型、热力热力学模型、物种特异性响应模型
- **Web 框架**：Flask
- **前端**：HTML、CSS

### 数据流
1. 用户选择站点和日期
2. 系统获取气象数据和 eBird 数据
3. 运行预测模型计算迁徙适宜度
4. 生成详细报告
5. 在 Web 页面展示结果
